HTML
HTML stands for Hyper Text Markup Language HTML is the standard markup language for creating Web pages
HTML element
An HTML element is defined by a start tag, some content, and an end tag:
HTML tag

Angle bracket
left (<) and right (>) angle brackets are used in HTML to identify the opening and closing tags of elements.
HTML attribute
All HTML elements can have attributes Attributes provide additional information about elements Attributes are always specified in the start tag Attributes usually come in name/value pairs like: name="value"
HTML comment
The comment tag is used to insert comments in the source code. Comments are not displayed in the browsers. You can use comments to explain your code, which can help you when you edit the source code at a later date. This is especially useful if you have a lot of code.
HTML object element
The tag defines a container for an external resource. The external resource can be a web page, a picture, a media player, or a plug-in application
HTML special character
